# EDB业务版图介绍培训演示稿 - 推荐大纲

## 推荐页数与时间分配

总时长：60 分钟  
推荐页数：41 页  
定位：成熟数据平台的官方培训演示，面向业务、产品、数据相关同事，兼顾概念认知、业务判断、数据治理和 SQL 实战。

| 模块 | 页数 | 时间 | 讲解目标 |
|---|---:|---:|---|
| 开场与导航 | 1-6 | 8 min | 建立培训目标、导航目录、EDB 产品定位和四大业务版图 |
| EDB 数据结构 | 7-16 | 19 min | 讲清 1.0 套表、1.0/2.0 对照、2.0 套表形式、维度体系和合规授权 |
| 宏观与地方经济 | 17-28 | 16 min | 建立宏观数据框架、全量来源地图、部委使用指南、常见口径问题和地方发布节奏 |
| 国际宏观与行业经济 | 29-35 | 8 min | 说明外部宏观联动、行业数据中观价值、行业覆盖与产业链标签 |
| 经营数据 | 36-39 | 5 min | 讲清经营数据边界、产品体系、标准化结构和业务框架 |
| SQL 实战与收束 | 40-41 | 5 min | 把查询场景设计成操作指南，最后以 SQL 操作台收束 |

## 页面结构

1. EDB业务版图介绍：封面，强视觉冲击
2. 60分钟训练地图：目录，点击跳转章节
3. 学完应该能回答什么：培训目标
4. EDB的产品定位：从分散数据到结构化时间序列产品
5. 四大业务模块：全国、区域、海外、产业
6. 如何判断数据所属业务：模块判断逻辑
7. 章节页：数据结构
8. EDB1.0套表结构：指标表、数据表、树形表、维度表、常量表
9. EDB1.0模块套表映射：四大模块表关系
10. EDB1.0 / 2.0结构对照：清晰对比
11. EDB2.0套表形式：1.2章节第一张表
12. EDB2.0模块套表映射：1.2章节第二张表
13. EDB2.0为什么升级：从采集到治理
14. EDB2.0维度体系：五组维度卡片
15. 15个维度总览：分组卡片，可展开查看
16. 合规授权：授权类型说明与样例截图
17. 章节页：宏观数据
18. 宏观数据的分析框架：增长、价格、政策、金融条件、资产
19. 宏观传导链条：可视化连线图，展示政策到实体再到验证指标
20. 中国宏观常见数据源：按 2.1 表格列出全部官方与市场来源
21. 国家统计局使用指南：发布日程、数据库、年鉴、公报、普查
22. 财政部使用指南：财政数据、决算专题、搜索入口与预算口径
23. 中国人民银行使用指南：统计数据、信息公开、搜索与社融结构
24. 海关总署使用指南：统计快讯、统计月报、在线查询与修订
25. 发改委使用指南：月度统计、PPP/项目、价格与政策信号
26. 宏观常见疑问：GDP、CPI、PPI、PMI、社融等
27. 地方经济发布节奏：预算、统计公报、决算、年鉴与发布特点
28. 地方经济常见口径问题：债务、人口、GDP、同比计算
29. 国际宏观：外需、汇率、利率、商品与风险偏好
30. 国际宏观数据源：国际组织、央行、统计机构、商品来源
31. 章节页：行业经济
32. 行业经济：连接宏观总量与公司基本面
33. 行业研究的五个问题：需求、供给、价格、库存、外需
34. 行业覆盖与来源导航：点击行业查看指标概览和数据来源
35. 行业特有数据标签：产品类别、上下游关系和指标类型标签
36. 章节页：经营数据
37. 经营数据产品体系：经营数据边界、三类表单与特殊范围
38. 经营数据标准化结构：公司 + 指标 + 维度 + 宽表示例
39. 经营数据业务框架：通用框架、行业属性框架、聚源纳入逻辑
40. 章节页：SQL实战
41. SQL实战操作台：定位指标、地区覆盖、经营数据查询，按钮切换案例

- `T`：在 `aurora / glassmorphism / tokyo-night / dracula` 之间切换主题。
- 目录卡片、章节按钮、下一步/返回按钮：点击跳转到对应页或章节。
- 维度卡片、合规卡片、SQL案例：点击切换展开状态。
- 27 张嵌入图片：按 Word 中对应章节分布到内容页中，点击进入放大层，再点击或按 `Esc` 还原。
