EDB Training Deck for Windows

How to open
1. Extract the whole zip to a local folder.
2. Double-click `Open-EDB-Training-Deck.cmd`, or open `index.html` directly in a browser.
3. Recommended browsers: Chrome or Edge.

Important
- Keep the folder structure unchanged. `index.html`, `assets`, `style.css`, and `interaction.js` must stay together.
- Do not open the HTML from inside the zip file preview. Extract first.

Presentation controls
- `S`: presenter mode
- `T`: switch theme
- `Esc`: close zoomed image overlay

Main entry
- `index.html`
